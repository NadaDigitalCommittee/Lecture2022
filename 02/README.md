## 課題
上の必須項目を読み、HTMLだけで下の画像のようなページを作ってください。また、ファビコンは下にあるデジ委のロゴを保存して使用すること。

文章の中身は完全に再現する必要は無いです　同じ感じの長さなら"あああああ"など適当な文字列でもOKです

提出方法は前回と同じです。
